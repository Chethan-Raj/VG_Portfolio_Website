# Images Folder

Place your image files here. Supported formats: .jpg, .jpeg, .png, .webp, .avif

## Required Files

| File | Used In | Recommended Size |
|------|---------|-----------------|
| `hero.jpg` | Hero section background | 1920×1080px min |
| `about.jpg` | About section | 800×1000px |

## Portfolio Images (place in `/portfolio/` subfolder)

| File | Portfolio Card |
|------|---------------|
| `portfolio/rare-gemstones.jpg` | Rare Gemstones |
| `portfolio/custom-jewelry.jpg` | Custom Jewelry |
| `portfolio/gem-auctions.jpg` | Gem Auctions |
| `portfolio/gemstone-sourcing.jpg` | Gemstone Sourcing |
| `portfolio/investment-consultancy.jpg` | Investment Consultancy |

All images fall back to beautiful color gradients if files are missing.
